Personal Introduction Program – Python

Project Overview
This project is a beginner-friendly Python program that collects user information and displays a personalized welcome message.

Objectives
- Learn Python basics
- Understand input/output
- Use variables and f-strings
- Practice running Python programs

 Setup Instructions
1. Install Python 3.x
2. Download this repository
3. Run the program using:
   python personal_intro.py

Project Structure
- personal_intro.py – Main Python program
- README.md – Project documentation
- requirements.txt – Dependency info
- screenshot.png – Program output screenshot

What I Learned
- How to use input() and print()
- How variables store data
- How Python executes code step by step
- How to use GitHub for submissions

Sample Output
Welcome message with user details displayed in a friendly format.

Testing
Tested with different names, ages, and hobbies. Program works without errors.
